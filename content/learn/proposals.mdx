## Proposal Submission Commands

### Text Proposal

<div className="p-6 border border-neutral-200 dark:border-neutral-800 rounded-lg mb-6">
  <div className="flex items-center gap-3 mb-4">
    <IconFileText className="h-5 w-5 text-neutral-500 dark:text-neutral-400" />
    <h4 className="text-lg font-semibold text-neutral-800 dark:text-neutral-200">Basic Text Proposal</h4>
  </div>
  <p className="text-neutral-600 dark:text-neutral-300 mb-4 text-sm">Submit a proposal for community discussion and signaling without on-chain execution.</p>
  ```bash copy seid tx gov submit-proposal \ --title "Community Initiative Proposal" \ --description "Detailed description of the initiative..." \ --type Text \ --deposit 50000000usei \ --from mykey \ --chain-id pacific-1 ```
</div>

## Proposal Management

### Query Proposals

  <div className="grid grid-cols-1 lg:grid-cols-2 gap-6 mb-8">
    
    <div className="p-5 border border-neutral-200 dark:border-neutral-800 rounded-lg">
      <h4 className="font-semibold text-neutral-800 dark:text-neutral-200 mb-3 flex items-center gap-2">
        <IconEye className="h-4 w-4 text-neutral-500 dark:text-neutral-400" />
        View Proposal Details
      </h4>
      ```bash copy
      # Get proposal by ID
      seid query gov proposal 1

      # View proposal status
      seid query gov proposals --status voting_period

      # Find proposer
      seid query gov proposer 1
      ```
    </div>

    <div className="p-5 border border-neutral-200 dark:border-neutral-800 rounded-lg">
      <h4 className="font-semibold text-neutral-800 dark:text-neutral-200 mb-3 flex items-center gap-2">
        <IconClock className="h-4 w-4 text-neutral-500 dark:text-neutral-400" />
        Track Voting Progress
      </h4>
      ```bash copy
      # Current vote tally
      seid query gov tally 1

      # Your vote status
      seid query gov vote 1 $(seid keys show mykey -a)

      # All votes
      seid query gov votes 1
      ```
    </div>

  </div>

### Weighted Voting

  <div className="p-6 border border-neutral-200 dark:border-neutral-800 rounded-lg mb-8">
    <div className="flex items-center gap-3 mb-4">
      <IconNote className="h-5 w-5 text-neutral-500 dark:text-neutral-400" />
      <h4 className="text-lg font-semibold text-neutral-800 dark:text-neutral-200">Split Your Vote</h4>
    </div>
    <p className="text-neutral-600 dark:text-neutral-300 mb-4 text-sm">Distribute your voting power across multiple options. Weights must total 1.0.</p>
    
    ```bash copy
    # Define vote weights
    voting_weights="yes=0.6,abstain=0.3,no=0.1"

    # Submit weighted vote
    seid tx gov weighted-vote 1 $voting_weights \
      --from mykey \
      --chain-id pacific-1
    ```

    <div className="mt-4 p-4 bg-neutral-50 dark:bg-neutral-900/50 rounded-lg">
      <h5 className="font-medium text-neutral-800 dark:text-neutral-200 mb-2 text-sm">Common Weight Patterns:</h5>

          <span className="font-mono text-neutral-600 dark:text-neutral-400">yes=0.7,abstain=0.3</span>
