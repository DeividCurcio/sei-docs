# Getting Started with Sei

## System Requirements

### Hardware Specifications

See our [System Configuration Guide](./node-configuration#hardware-tuning) for detailed performance tuning.

| Component | Minimum      | Recommended  |
| --------- | ------------ | ------------ |
| CPU       | 8 cores      | 16 cores     |
| RAM       | 32GB         | 64GB         |
| Storage   | 1TB NVMe SSD | 2TB NVMe SSD |
| Network   | 1Gbps        | 2Gbps        |

### Base System

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install make gcc git jq chrony curl lz4 wget -y
```

See [Security Best Practices](./best-practices#system-security) for post-installation hardening steps.

## Installation Options

### 1. Pre-built Binary

[Download latest release](https://github.com/sei-protocol/sei-chain/releases)

```bash
wget -O seid <binary_url_for_version>
sudo mv seid /usr/local/bin/
sudo chmod +x /usr/local/bin/seid
```

### 2. Quick Start Script

For testing or development environments. See [Local Development Guide](../develop/local-testnet) for more options.

```bash
git clone https://github.com/sei-protocol/sei-chain
cd sei-chain
python3 scripts/run-node.py
```

### 3. Build from Source

For production environments, see our [Node Operators Guide](./node-operators#building-from-source) for detailed build instructions.

## Basic Configuration

See [Configuration Reference](./node-configuration#configuration-files) for complete documentation.

### Node Directory Structure

```bash
~/.sei/config/
├── app.toml          # Application settings
├── client.toml       # CLI configuration
├── config.toml       # Tendermint settings
├── genesis.json      # Chain genesis file
├── node_key.json     # P2P authentication
└── priv_validator_key.json  # Validator key (if applicable)
```

### Essential Settings

```toml
# app.toml
[state-commit]
sc-enable = true    # Enable sei-db for better performance

[state-store]
ss-enable = true    # Enable historical state

minimum-gas-prices = "0.01usei"  # Minimum transaction fees
```

See [Database Optimization](./node-configuration#database-settings) for performance tuning.

### Default Ports

See [Network Configuration](./node-configuration#network-settings) for security recommendations.

- P2P: 26656
- RPC: 26657
- API: 1317
- gRPC: 9090

## Service Setup

For detailed service management, see [Node Operations](./best-practices#service-management).

```ini
# /etc/systemd/system/seid.service
[Unit]
Description=Sei Node
After=network.target

[Service]
User=$USER
ExecStart=$(which seid) start
Restart=always
RestartSec=3
LimitNOFILE=65535

[Install]
WantedBy=multi-user.target
```
